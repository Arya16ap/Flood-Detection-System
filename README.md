# FLOOD_DETECTIONS_PYTHON_V1

This was a project i made for a competition, where the general theme was find a basic solution to problems:)

I ended up with this:
so I realized, that during rainfall, a lot of people miss their college attendance, or go to their work late, as they arent aware which lanes, roads and all are flooded.

Our project uses the intensity of rainfall, and the low lying areas as 2 constraints, and simulates a flood in such areas.

This has many applications. the one we chose is to find a route from home to your work place, wihtout any interruptions from flood. this Project has a lot of potential.
